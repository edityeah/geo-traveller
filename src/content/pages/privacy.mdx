Geo-Traveller is a personal travel journal. This page describes what data the site collects.

### What we collect

The site is a static site served from Cloudflare's CDN. We do not run a server-side database, set tracking cookies, or collect personal information from visitors.

### Analytics

If analytics is enabled, we use privacy-friendly analytics that collect aggregate visit counts only — no IP addresses, no fingerprinting, no cookies.

### Comments and newsletter

If you choose to leave a comment (via Giscus, backed by GitHub Discussions) or subscribe to the email newsletter, the respective third party — GitHub or the newsletter provider — handles your data per their own privacy policies.

### Contact

Questions? Use the [contact page](https://geo-traveller.com/contact/).
